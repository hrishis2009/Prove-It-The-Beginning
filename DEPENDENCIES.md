# Dependencies

See [`preamble.tex`](https://github.com/hrishis2009/Prove-It-The-Beginning/blob/main/src/preamble.tex) and [`main.tex`](https://github.com/hrishis2009/Prove-It-The-Beginning/blob/main/src/main.tex).

### tcolorbox Libraries

- `skins`

## Notes
A standard installation of [TeX Live](https://tug.org/texlive/), [MacTeX](https://www.tug.org/mactex/), or [MiKTeX](https://miktex.org/) should provide all required packages. 

Compilation has been tested primarily using `latexmk` with pdfLaTeX on [MacTeX](https://www.tug.org/mactex/), as well as on [Overleaf](https://www.overleaf.com/). Users of other LaTeX distributions, compilers, or build systems may need to manually install dependencies or make minor adjustments to the build process.
